FAQ:

What platforms are supported?

We currently support Windows (7+), MacOS (Monterey+), Debian GNU/Linux and Fedora GNU/Linux.
We also have plans to bring support to FreeBSD.

Does Atomic contain telemetry?

No. Atomic will never store any user data, whether it be anonymized or not.
Atomic is a free software project, and collecting user data would be violation of user freedoms.

Does Atomic have syntax highlighting?

Yes! We support syntax highlighting for most major programming languages.

Does Atomic have an integrated terminal?

Yes, Atomic has a live integrated terminal.
